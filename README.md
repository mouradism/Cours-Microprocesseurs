# Cours Microprocesseurs

This repository contains course materials and projects for microprocessors and computer architecture.

## Contents

### Projects

- **8-bit Computer** (`dev/8bitComputer/`) - Custom 8-bit computer design and simulation
- **Simple Computer Projects** (`simplecomp*.pdsprj`) - Various computer architecture simulations
- **RAM Testing** (`ramTEST.pdsprj`) - Memory testing and verification projects
- **Flip-Flop Circuits** (`jk.pdsprj`, `jsbas*.pdsprj`) - Digital logic circuits

### Additional Projects

- **MouNes** (`MouNes/`) - NES emulator implementation with SFML graphics library
- **Development Tools** (`dev/`) - Supporting tools and resources

## Tools & Technologies

- **Proteus Design Suite** - Circuit design and simulation
- **SFML 2.0/2.5.1** - Graphics and multimedia library
- **C/C++** - Programming language

## File Structure

```
├── dev/

```
